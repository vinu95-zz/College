1ST YEAR LABORATORY

Computer Programming Lab

Objectives:- Dennis Richards introduced ‘C’ language and it has its major role in the development of packages. Simple example like quadratic equation, GCD & LCM, reversing a string, palindrome are verified for a sample of inputs. Parallel programming concepts are presented very dramatically.

2ND YEAR LABORATORY

Data Structure / Object Oriented Programming (OOPS) Lab

Objectives:- Dennis Richards introduced ‘C’ language and it has its major role in the development of packages. Simple example like quadratic equation, GCD & LCM, reversing a string, palindrome are verified for a sample of inputs. Parallel programming concepts are presented very dramatically.

Electronic Circuit & Logic Design Lab

Objectives:- Electronic Circuits existence is mandatory in the day-to-day operations without which functioning of some components becomes impossible. Capacitor, resistor, zener diode, wall shaping circuits, multi vibrators, transistors, oscillators, operational amplifiers Schmitt trigger are effectively utilized and working of which are analyzed successfully. Logic design experiments like decorders, multiplexes, shifting of bits, counters, up & down counters are successfully verified. Pspiu & Link are two softwares used to simulate the above applications.

3RD YEAR LABORATORY

SS/OS Lab

Objectives:- System software and operating System Lab consists of 12 different programs, where this lab includes LEX, YACC and Operating System Programs. this Lab will be run in Ubuntu open source operating system.

Database Management Lab

Objectives:- Database Management System Lab Consists of 5 different Programs. In this Lab Students can be able to learn database concepts, how to store data into database and retrieving the data from database. The tool used for running this program is SQL+ and students also will be able to run queries with structured format.

4TH YEAR LABORATORY

Programming the Web Lab

Objectives:- Programming the Web Lab Contains 12 different Programs prescribed by VTU, from this students can able to achieve,
  • The World Wide Web – protocols (HTTP) and architecture
  • Dynamic web pages – protocols and methods to support dynamic web pages (browser-side methods, server-side methods)
  • Web programming languages – the popular programming languages and techniques used on the web
  • Data transfer methods over the web – the current methods for data transfer
  • Security issues of the web environment
  • We will do the Web lab in Ubuntu Operating Systems
